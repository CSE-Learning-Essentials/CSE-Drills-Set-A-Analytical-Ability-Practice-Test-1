Direction: Each question below consists of a related pair of words. From among the suggested answers, choose the word or group of words that best expresses a similar relationship from the original pair.

1. MARTILYO : PAKO :: WRENCH : ____

2. APOY : SUNOG :: ULAN : ____

3. DALIRI : KAMAY :: GULONG : ____

4. BANDILA : BANSA :: KRUS : ____

5. ARAW : MAINIT :: YELONG TUBIG : ____

6. KATATAWANAN : TUWA :: TRAHEDYA : ____

7. GABI : DILIM :: UMAGA : ____

8. LAPIS : PAPEL :: BRUSH : ____

9. GINTO : MAHALAGA :: TUBIG : ____

10. SINGSING : KASAL :: BATA : ____

Correct Answers & Explanation:

  1. B. Bolt — Tool : Object it acts on
  2. A. Baha — Cause : Effect
  3. A. Sasakyan — Part : Whole
  4. A. Iglesia — Symbol : What it represents
  5. B. Malamig — Object : Characteristic
  6. B. Lungkot — Genre or Event : Emotion it causes
  7. C. Maliwanag — Time : Typical condition
  8. D. Canvas — Tool : Surface it is used on
  9. D. Mahalaga — Object : Value or Quality
  10. E. Pamilya — Object : What it symbolizes or leads to
